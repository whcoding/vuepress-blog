### Redis 入门

> Redis 是什么

REmote DIctionary Server(Redis) 是一个由 Salvatore Sanfilippo 写的 key-value 存储系统，是跨平台的非关系型数据库.

> Redis 能干嘛?

1. 内存存储, 持久化,内存中是断电即失, 所以持久化很重要(rdb, aof)
2. 效率高, 可用于高速缓存
3. 订阅发布系统
4. 地图信息分析
5. 计时器, 计数器...等等

> 特性
1. 多样的数据类型
2. 持久化
3. 集群
4. 事务
