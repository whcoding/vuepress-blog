## redis key 操作

### Redis-Key
- 判断当前key是否存在

<!-- more -->

```
exists key
```
- 删除key
```
del key
```

- 移动当前key到其他的db
```
move key db
```

- 设置key过期时间 单位是秒
```
expire key 10
```

- 查看当前key剩余时间
```
ttl key
```





- 查看当前key的类型
```
type key
```
