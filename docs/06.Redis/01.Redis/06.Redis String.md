### redis string 操作


- 设置一个key的value值
```
set name xiaoming
```

- 获取一个key的value值
```
get key
```

- 追加一个值到key上
```
append key value
```

- 截取字符串
```
getrange key start end # start 开始位置, end 结束位置
```


<!--more-->


- 替换字符串
```
setrange key offset value # offset 要替换的位置, value 替换的值
```

- 设置多个值
```
mset k1 k2 k3 v1 v2 v3
```

- 获取多个值

```
mget v1 v2 v3 
```
