### mysql binlog 



### binlog 常用命令

1. 查看是否开启

```mysql
show variables like 'log_%'; 
```

2. 查看当前使用的biglog 日志 

```mysql
show master status;
```

3. 查看全部biglog日志

```mysql
show master LOGS;
```

4. 刷新 binlog 日志. 产生一个新的文件

```mysql
flush logs;
```

5. 重置(清空)所有 binlog 日志

```mysql
reset master;
```



### 开启binlog

- 在my.cnf中加入

````shell
[mysqld]
server_id = 1
log_bin = /var/log/mysql/mysql-bin.log
expire_logs_days = 1
max_binlog_size  = 100M
````
