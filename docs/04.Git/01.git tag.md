### 本地创建 tag 详细版(常用)
```
git tag -a v1.0 -m '版本描述'
```

#### 轻量 tag
```
git tag 
```

### 显式的推送 tag 
```
git push origin v1.0
```

### 远端拉取
```
git fetch --tags  # 拉取所有tag
git checkout -b v1.0 refs/tags/v1.0
```

#### 精准拉取某个tag
```
git fetch origin tag v1.0
```


### 查看 tag 详细信息
```
git show v1.0
```
